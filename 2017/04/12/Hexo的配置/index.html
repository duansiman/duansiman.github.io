文章中插入图片

　在博客的_config.yml文件设置post_asset_folder为true。在建立文件时，Hexo会自动建立一个与文章同名的文件夹，用来放资源的。
然后安装hexo-asset-image插件npm install hexo-asset-image –save。通过下面方法就可以在文章插入图片。
