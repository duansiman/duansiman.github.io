安逸的小猫

苟有恒，何必三更眠、五更起；最无益，莫过一日曝、十日寒。

2017-04-18
English

句子成分

词性担当

主语——名、代（主格）
谓语——动词
表语——名、代、形、副(少量)、介宾
宾语——名、代（宾格）

定语

修饰名词，对名词进行限定的语言

形容词做定语：形容+名

a clean room

介宾做定语：名+介宾

the box on the table 桌子上的盒子

部分副词做定语：名+副词

children there 哪里的小孩

状语

修饰动词，介宾、副词担当，放在句首、句尾和句中都可以
